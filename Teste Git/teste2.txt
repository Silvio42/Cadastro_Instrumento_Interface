-- Para entrar em um repositorio deve usar o comando cd, e se o nome da pasta tiver espaço usar o " " (aspas duplas) no caminho da mesma

-- dica do curso trocar onde estiver "master" para "main"

Comandos:

"git init" -> cria o repo git no seu diretorio local

"git status" -> Mostra como está o estado do seu fonte local o comparando com o git na nuvem

"git add <NomeDoArquivo>" -> Adiciona o arquivo ao projeto, podemos usar com o caminho do diretório para adicionar tudo que está ao diretório, também podemos usar o "git add ." para estar adicionando tudo

"git commit" possui duas flags "-a" para enviar todos os arquivos e o "-m" para enviar uma mensagem, o "-m" deve estar com um texto assim logo após "exemplo do git de comentário"

"git pull" (pull - puxar) da um update na sua base local para ficar igual ao repo na nuvem

"git push" (push - empurrar) envia os commits de seu repo para o repo do github na nuvem

"git remote" add origin https://github.com/Silvio42/Cadastro_Instrumento_Interface.git -> Indicar o repo da nuvem para seu repo local

"git push --set-upstream origin master " -> Enviar o push para o git sem ter criado a brach master
